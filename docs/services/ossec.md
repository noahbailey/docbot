# OSSEC HIDS

### Prerequisites

    sudo apt install gcc make libevent-dev zlib1g-dev  libssl-dev libpcre2-dev wget tar -y


### Install from source

    wget https://github.com/ossec/ossec-hids/archive/3.6.0.tar.gz
    tar xzf 3.6.0.tar.gz
    cd ./ossec-hids-3.6.0

Build and install: 

    sudo ./install.sh
